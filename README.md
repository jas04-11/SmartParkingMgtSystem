# 🚗 Smart Parking Management System

## 📌 Overview
The **Smart Parking Management System** is a console-based application developed using **C++**.  
It helps in managing parking spaces efficiently by providing features like user authentication, vehicle parking, searching, and removal.

The system also includes an **Admin Panel** to monitor parking activity and track total earnings.

---

## 🎯 Features
- 🔐 User Registration & Login  
- 🅿️ Automatic Parking Slot Allocation  
- 📊 Real-Time Parking Layout Display  
- 🚗 Park & Remove Vehicles  
- 🔍 Search Vehicle by Number  
- ⏱️ Time-Based Fee Calculation  
- 👨‍💼 Admin Panel  
- 💰 Total Earnings Tracking  
- 💾 File Handling for Data Storage

## ⚙️ Installation & Run
# Clone the repository
git clone https://github.com/your-username/smart-parking-system.git

# Navigate to project folder
cd smart-parking-system

# Compile the program
g++ main.cpp -o parking

# Run the program
./parking
